◆ empate()

bool empate ( vector< vector< int > >  tabuleiro)

Verifica se o jogo terminou em empate.

Considera que o jogador X sempre inicia, então empate ocorre com 5 jogadas de X e 4 de O, sem que nenhum tenha vencido.

Parâmetros
tabuleiroMatriz 3x3 representando o estado do jogo.
Retorna
true se o jogo terminou empatado, false caso contrário.

◆ impossivel()

bool impossivel ( vector< vector< int > >  tabuleiro)

Verifica se o estado atual do jogo é impossível.

Situações impossíveis incluem: X jogou mais de uma vez a mais que O, ou O jogou mais que X.

Parâmetros
tabuleiroMatriz 3x3 representando o estado do jogo.
Retorna
true se o estado do tabuleiro é impossível, false caso contrário.
